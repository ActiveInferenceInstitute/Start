# Translated Curriculums Technical Reference

## Overview

Technical documentation for translated curriculum file format and structure.

## Directory Structure

Organized by language subdirectories:
- Each language has its own subdirectory (lowercase)
- Files are named: `{entity}_curriculum_{language}_{timestamp}.md`

## File Format

### Markdown with YAML Frontmatter

```markdown
---
title: "Curriculum Title"
entity: "entity_name"
language: "target_language"
translation_date: "YYYY-MM-DD HH:MM:SS"
---

# Translated Curriculum Content
...
```

### File Naming

**Pattern**: `{entity}_curriculum_{language}_{timestamp}.md`

**Examples**:
- `Barry Bonds_curriculum_chinese_20250903_084530.md`
- `Coffee Roasting_curriculum_spanish_20250903_083403.md`

## Generation

Files are generated by:
- `learning/curriculum_creation/4_Translate_Introductions.py`
- `src/perplexity/translation.py::process_translations()`
