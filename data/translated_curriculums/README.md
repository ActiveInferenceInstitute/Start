# Translated Curriculums

Translated curriculum outputs organized by target language.

## Overview

This directory contains translated versions of curricula, organized into language-specific subdirectories.

## Structure

```
translated_curriculums/
├── chinese/
│   └── {entity}_curriculum_chinese_{timestamp}.md
├── spanish/
│   └── {entity}_curriculum_spanish_{timestamp}.md
├── french/
│   └── {entity}_curriculum_french_{timestamp}.md
└── ...
```

## File Format

**Format**: Markdown with YAML frontmatter
**Naming**: `{entity}_curriculum_{language}_{timestamp}.md`

## Languages

Supported languages include:
- Chinese (Simplified)
- Spanish
- Arabic
- Hindi
- French
- Japanese
- Russian
- Swahili
- Tagalog
- Ancient Greek
- And more (see `data/config/languages.yaml`)
