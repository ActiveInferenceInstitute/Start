# Domain Research Technical Reference

## Overview

Technical documentation for domain research output files.

## File Formats

### JSON Schema

```json
{
  "timestamp": "string (YYYYMMDD_HHMMSS)",
  "domain_name": "string",
  "domain_analysis": "string (markdown content)",
  "curriculum_content": "string (markdown content)",
  "processing_time": "string"
}
```

### Markdown Format

Human-readable research report with same content as JSON, formatted for reading.

## File Naming

**Pattern**: `{domain}_research_{timestamp}.json` and `.md`

**Examples**:
- `Coffee Roasting_research_20250903_073951.json`
- `Coffee Roasting_research_20250903_073951.md`
- `Myrmecology_research_20250903_080643.json`

## Generation

Files are generated by:
- `learning/curriculum_creation/1_Research_Domain.py`
- `src/perplexity/domain.py::analyze_domain()`
