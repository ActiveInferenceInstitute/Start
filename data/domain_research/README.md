# Domain Research Outputs

Domain research outputs from Perplexity API analysis.

## Overview

This directory contains research outputs for professional domains, generated by the domain research pipeline.

## File Formats

**Formats**: JSON and Markdown
**Naming**: `{domain}_research_{timestamp}.json` and `.md`

## File Structure

JSON contains:
- Timestamp
- Domain name
- Domain analysis
- Curriculum content
- Processing time

Markdown contains human-readable research report.

## Usage

These files are used as input for curriculum generation in `2_Write_Introduction.py`.
