# Audience Research Outputs

Entity/audience research outputs from Perplexity API analysis.

## Overview

This directory contains research outputs for target audiences/entities, generated by the entity research pipeline.

## File Format

**Format**: JSON
**Naming**: `{entity}_research_{timestamp}.json`

## File Structure

```json
{
  "timestamp": "20250903_073951",
  "entity_name": "karl_friston",
  "entity_description": "Description text",
  "research_data": "Comprehensive research analysis...",
  "processing_time": "45.2 seconds"
}
```

## Usage

These files are used as input for curriculum generation in `2_Write_Introduction.py`.
