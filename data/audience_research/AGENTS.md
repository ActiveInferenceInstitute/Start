# Audience Research Technical Reference

## Overview

Technical documentation for audience research output files.

## File Format

### JSON Schema

```json
{
  "timestamp": "string (YYYYMMDD_HHMMSS)",
  "entity_name": "string",
  "entity_description": "string",
  "research_data": "string (markdown content)",
  "processing_time": "string"
}
```

### Fields

- `timestamp`: ISO format timestamp of research generation
- `entity_name`: Name of researched entity/audience
- `entity_description`: Description of entity
- `research_data`: Comprehensive research analysis in markdown format
- `processing_time`: Processing time as string

## File Naming

**Pattern**: `{entity}_research_{timestamp}.json`

**Examples**:
- `Barry Bonds_research_20250903.json`
- `Bill Gates_research_20250903.json`
- `Elon Musk_research_20250903.json`

## Generation

Files are generated by:
- `learning/curriculum_creation/1_Research_Entity.py`
- `src/perplexity/entity.py::research_target_audience()`
