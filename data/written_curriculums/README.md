# Written Curriculums

Generated curriculum outputs organized by entity/domain.

## Overview

This directory contains complete generated curricula, organized into entity/domain-specific subdirectories.

## Structure

```
written_curriculums/
├── {entity}/
│   ├── {section}_{timestamp}.md
│   ├── complete_curriculum_{timestamp}.md
│   └── {entity}_metadata_{timestamp}.json
└── ...
```

## File Formats

**Formats**: Markdown (.md) and JSON
**Naming**:
- Sections: `{section}_{timestamp}.md`
- Complete: `complete_curriculum_{timestamp}.md`
- Metadata: `{entity}_metadata_{timestamp}.json`

## Content

Curricula include:
- Comprehensive learning modules (40-60 hours)
- Section-by-section breakdowns
- Assessment materials
- Metadata and metrics
