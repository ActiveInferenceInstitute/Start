# Written Curriculums Technical Reference

## Overview

Technical documentation for written curriculum file formats and structure.

## Directory Structure

Organized by entity/domain subdirectories:
- Each entity/domain has its own subdirectory
- Contains section files, complete curriculum, and metadata

## File Formats

### Section Files

**Format**: Markdown
**Naming**: `{section}_{timestamp}.md`

**Content**: Individual curriculum section with markdown formatting

### Complete Curriculum

**Format**: Markdown
**Naming**: `complete_curriculum_{timestamp}.md`

**Content**: Complete curriculum with all sections concatenated

### Metadata Files

**Format**: JSON
**Naming**: `{entity}_metadata_{timestamp}.json`

**Schema**:
```json
{
  "entity_name": "string",
  "timestamp": "string",
  "sections": ["section1", "section2", ...],
  "total_words": number,
  "total_sections": number
}
```

## Generation

Files are generated by:
- `learning/curriculum_creation/2_Write_Introduction.py`
- `src/perplexity/curriculum.py::process_research_file()`
