# Learning Pipeline Technical Reference

## Overview

Technical documentation for the learning pipeline structure and workflows.

## Directory Structure

### `curriculum_creation/`
Contains curriculum creation scripts:
- `1_Research_Domain.py`: Domain research
- `1_Research_Entity.py`: Entity research
- `2_Write_Introduction.py`: Curriculum generation
- `3_Introduction_Visualizations.py`: Visualization generation
- `4_Translate_Introductions.py`: Translation
- `generate_curriculum_gui.py`: GUI generator
- `generate_custom_curriculum.py`: Custom curriculum generator

## Pipeline Flow

1. Research → 2. Content → 3. Visualization → 4. Translation
