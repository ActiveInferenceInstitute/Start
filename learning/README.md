# Learning Pipeline

Curriculum creation scripts and workflows.

## Overview

This directory contains scripts and utilities for the curriculum creation pipeline, from research to translation.

## Structure

### `curriculum_creation/`
Main curriculum creation scripts:
- Research scripts (domain and entity)
- Content generation scripts
- Visualization scripts
- Translation scripts
- GUI and custom curriculum generators

## Pipeline Stages

1. **Research**: Domain and entity analysis
2. **Content Generation**: Curriculum creation
3. **Visualization**: Charts and diagrams
4. **Translation**: Multilingual content
