# VFE Technical Reference

## Overview

Technical documentation for the Variational Free Energy (VFE) perception simulation example.

## Files

### `vfe.jsx`
React/JSX component file demonstrating Variational Free Energy perception simulation.

**Purpose**: Interactive visualization of Variational Free Energy (VFE) mathematical framework for perception, including:
- Gaussian generative model (prior and likelihood)
- Recognition model (approximate posterior)
- Free Energy vs Surprise relationship
- Interactive parameter adjustment
- Real-time visualization updates

**Framework**: React with JSX
**Dependencies**: 
- React 18.2.0
- ReactDOM 18.2.0
- lucide-react 0.294.0 (for icons)
- Tailwind CSS (via CDN in compiled version)

**Key Components**:
- `VFEDashboard` - Main component with state management
- `DistributionPlot` - SVG visualization of probability distributions
- `FreeEnergyLandscape` - F(μᵩ) landscape plot
- `TradeoffPlot` - Complexity vs Accuracy trade-off visualization
- `EnergyBars` - Bar chart of energy components
- `DynamicEquation` - Animated equation display with scaling
- `Section` - Collapsible section component

**Mathematical Functions**:
- `gaussianPDF` - Gaussian probability density function
- `klDivergence` - KL divergence between two Gaussians
- `negLogLikelihood` - Expected negative log likelihood
- `computeOptimalPosterior` - Bayesian posterior computation
- `computeSurprise` - Surprise (negative log evidence)
- `computeFAtMu` - Free Energy at arbitrary μᵩ
- `validateComputation` - Mathematical validation checks

### `vfe-compiled.html`
Standalone HTML file compiled from `vfe.jsx`.

**Purpose**: Self-contained demonstration that can be opened directly in a browser without build tools or dependencies.

**Format**: Single HTML file with:
- Embedded JavaScript (bundled React, ReactDOM, component code)
- Tailwind CSS via CDN
- All dependencies included

**Usage**: Open directly in web browser

**Build**: Generated by `build-vfe.js` using esbuild

### `build-vfe.js`
Build script for compiling JSX to standalone HTML.

**Purpose**: Bundles React, ReactDOM, component code, and dependencies into a single HTML file.

**Process**:
1. Uses esbuild to bundle `vfe-entry.jsx`
2. Injects bundled code into HTML template
3. Outputs `vfe-compiled.html`

**Dependencies**: esbuild

### `vfe-entry.jsx`
Entry point for the build process.

**Purpose**: Imports the main component and makes React/ReactDOM globally available for the compiled HTML.

### `capture-gifs.js`
Automated GIF generation script using Playwright.

**Purpose**: Captures interactive demonstrations of the VFE visualization as animated GIFs by automating browser interactions.

**Dependencies**:
- Playwright 1.40.0 (browser automation)
- gifencoder 2.0.1 (GIF encoding)
- canvas 2.11.2 (image processing)

**Process**:
1. Launches headless Chromium browser via Playwright
2. Opens `vfe-compiled.html` from file system
3. Waits for React app to fully render
4. Performs automated interactions:
   - Slider movements (gradual value changes)
   - Section toggles (expand/collapse)
   - Manual mode activation and adjustment
   - Parameter combinations
5. Captures screenshots at intervals (100-200ms)
6. Converts screenshot sequences to animated GIFs
7. Saves GIFs to `gifs/` directory

**Scenarios Captured**:
- `slider-demo.gif` - Animated slider movements showing parameter changes
- `sections-demo.gif` - Section expansion/collapse animations
- `manual-mode-demo.gif` - Manual mode toggle and suboptimal belief exploration
- `parameter-combinations.gif` - Different parameter sets (balanced, high sensory trust, strong prior, conflicting signals)
- `full-demo.gif` - Complete interactive demonstration combining multiple interactions

**Technical Details**:
- Viewport: 1920x1080
- Frame rate: 8 fps
- GIF quality: 10 (default)
- Uses `file://` protocol to load local HTML
- Waits for React state updates after interactions
- Cleans up temporary screenshot files after GIF creation

**Usage**: `npm run capture-gifs`

**Requirements**: 
- `vfe-compiled.html` must exist (run `npm run build` first)
- Playwright browsers installed (`npx playwright install`)
