🧠 Variational Free Energy: Perception Simulation

An interactive React-based visualization of the VFE mathematical framework for perception! 

✨ Features:
• Gaussian generative model (prior & likelihood)
• Recognition model (approximate posterior)
• Free Energy vs Surprise relationship
• Real-time parameter adjustment
• Multiple visualization types (distributions, landscapes, trade-off plots)

🎬 Automated GIF Generation:
Run `npm run capture-gifs` to generate 5 animated demos showing:
- Slider movements & parameter changes
- Section expansions/collapses
- Manual mode exploration
- Different parameter combinations
- Full interactive walkthrough

🚀 Quick Start:
```bash
cd examples/vfe
npm install
npm run build
# Open vfe-compiled.html in browser
```

Built with React, Tailwind CSS, and automated with Playwright for GIF capture. Standalone HTML - no server needed! 🎯
