# EFE Technical Reference

## Overview

Technical documentation for the Expected Free Energy (EFE) gridworld action selection example.

## Files

### `efe.jsx`
React/JSX component file demonstrating Expected Free Energy action selection in a gridworld.

**Purpose**: Interactive visualization of Expected Free Energy (EFE) mathematical framework for action selection, including:
- Gridworld environment with Temperature and pH attributes
- Agent with probabilistic beliefs about tile values
- Action evaluation via Pragmatic and Epistemic values
- Interactive preference distribution controls
- Real-time EFE computation and visualization

**Framework**: React with JSX
**Dependencies**: 
- React 18.2.0
- ReactDOM 18.2.0
- lucide-react 0.294.0 (for icons)
- Tailwind CSS (via CDN in compiled version)

**Key Components**:
- `EFEDashboard` - Main component with state management
- `GridworldVisualization` - Grid display with tiles and agent position
- `PreferenceDistributions` - Temperature and pH preference controls
- `ActionEvaluationPanel` - EFE values and breakdowns for each action
- `BeliefVisualization` - Uncertainty heatmap and probability distributions
- `Section` - Collapsible section component

**Mathematical Functions**:
- `computeEFE` - Main EFE computation for an action
- `computePragmaticValue` - KL divergence between predicted outcome and preference
- `computeEpistemicValue` - Expected information gain from action
- `updateBeliefs` - Bayesian update of beliefs after observation
- `predictOutcome` - Determine which tile will be visited
- `entropy` - Compute entropy of probability distribution
- `klDivergence` - KL divergence between two distributions

### `efe-compiled.html`
Standalone HTML file compiled from `efe.jsx`.

**Purpose**: Self-contained demonstration that can be opened directly in a browser without build tools or dependencies.

**Format**: Single HTML file with:
- Embedded JavaScript (bundled React, ReactDOM, component code)
- Tailwind CSS via CDN
- All dependencies included

**Usage**: Open directly in web browser

**Build**: Generated by `build-efe.js` using esbuild

### `build-efe.js`
Build script for compiling JSX to standalone HTML.

**Purpose**: Bundles React, ReactDOM, component code, and dependencies into a single HTML file.

**Process**:
1. Uses esbuild to bundle `efe-entry.jsx`
2. Injects bundled code into HTML template
3. Outputs `efe-compiled.html`

**Dependencies**: esbuild

### `efe-entry.jsx`
Entry point for the build process.

**Purpose**: Imports the main component and makes React/ReactDOM globally available for the compiled HTML.

### `capture-gifs.js`
Automated GIF generation script using Playwright.

**Purpose**: Captures interactive demonstrations of the EFE visualization as animated GIFs by automating browser interactions.

**Dependencies**:
- Playwright 1.40.0 (browser automation)
- gifencoder 2.0.1 (GIF encoding)
- canvas 2.11.2 (image processing)

**Process**:
1. Launches headless Chromium browser via Playwright
2. Opens `efe-compiled.html` from file system
3. Waits for React app to fully render
4. Performs automated interactions:
   - Preference slider movements (Temperature and pH)
   - Action execution and agent movement
   - Tile selection in gridworld and uncertainty heatmap
   - Action prior adjustments
   - Beta parameter changes (posterior visualization)
   - Section toggles (expand/collapse)
   - Full workflow demonstrations
5. Captures screenshots at intervals (300-500ms)
6. Converts screenshot sequences to animated GIFs
7. Saves GIFs to `gifs/` directory

**Scenarios Captured**:
- `preference-sliders.gif` - Animated preference slider movements showing parameter changes
- `action-execution.gif` - Action sampling and agent movement through gridworld
- `tile-selection.gif` - Tile clicking and belief distribution exploration
- `action-prior.gif` - Action prior distribution adjustments
- `posterior-demo.gif` - Beta parameter changes and posterior visualization
- `sections-demo.gif` - Section expansion/collapse animations
- `full-demo.gif` - Complete interactive demonstration combining multiple interactions

**Technical Details**:
- Viewport: 1920x1080
- Frame rate: 8 fps
- GIF quality: 10 (default)
- Uses `file://` protocol to load local HTML
- Waits for React state updates after interactions
- Cleans up temporary screenshot files after GIF creation

**Usage**: `npm run capture-gifs`

**Requirements**: 
- `efe-compiled.html` must exist (run `npm run build` first)
- Playwright browsers installed (`npx playwright install`)
